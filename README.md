# Credit Risk Scoring System

## Overview
Built an end-to-end Credit Risk Scoring System using the Home Credit Default Risk dataset.

## Features
- Data Cleaning
- Feature Engineering
- Bureau Data Aggregation
- Previous Loan Analysis
- Installment Payment Analysis
- Logistic Regression
- Random Forest
- XGBoost
- Credit Score Generation
- Streamlit Dashboard

## Dataset
Home Credit Default Risk Dataset

## Results

| Model | ROC-AUC |
|---------|---------|
| Logistic Regression | 0.7574 |
| Random Forest | 0.7339 |
| XGBoost | 0.7685 |

## Top Features
- EXT_SOURCE_2
- EXT_SOURCE_3
- LATE_PAYMENT_RATE
- REFUSED_COUNT
- APPROVAL_RATE
