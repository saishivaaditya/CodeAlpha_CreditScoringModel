# CodeAlpha_CreditScoringModel
Dataset available from Kaggle Home Credit Default Risk Competition.
## Dataset

This project uses the Home Credit Default Risk dataset available on Kaggle.

Dataset Link:
https://www.kaggle.com/competitions/home-credit-default-risk

Due to dataset size and licensing considerations, raw CSV files are not included in this repository. Users can download the dataset directly from Kaggle and place the required files in the project data folder.


# Credit Risk Scoring System

## Overview
Built an end-to-end Credit Risk Scoring System using the Home Credit Default Risk dataset.

## Features
- Data Cleaning
- Feature Engineering
- Bureau Data Aggregation
- Previous Loan Analysis
- Installment Payment Analysis
- Logistic Regression
- Random Forest
- XGBoost
- Credit Score Generation
- Streamlit Dashboard

## Dataset
Home Credit Default Risk Dataset

## Results

| Model | ROC-AUC |
|---------|---------|
| Logistic Regression | 0.7574 |
| Random Forest | 0.7339 |
| XGBoost | 0.7685 |
